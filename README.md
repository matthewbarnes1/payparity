# PayParity

PayParity — EU Pay Transparency Directive readiness for mid-size employers.

Winner of the 50→10→3 idea funnel (Codeathon Round 4, July 2026). Whitepaper served live at `/whitepaper`.

## Run

```bash
npm install && npm start   # http://localhost:3022
```

Node >= 22.5, zero native dependencies, deploys to Vercel out of the box. MIT licensed.
